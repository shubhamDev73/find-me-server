Documentation

Root node: /api/

    Authorization end-points. All are POST requests.

  • /register/ : to register. data: username, password. returns "token": auth token
  • /login/ : to login. data: username, password. returns "token": auth token
  • /logout/ : to logout.

    User end-points. Require "Authorization: Bearer <auth-token>" as header. All are GET requests.

  • /personality/ : retrieve user personality.

All response are in json format and contain "error" field which contains the error message if an error occured, otherwise empty.
